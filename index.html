Quantities for Foodsby Orders

This webpage allows Foodsby restaurant partners to upload their daily orders and generate a list of the total quantity for each item.

Select 'All Meals' to view receipts / labels for all meals. Right-click in popup and select 'View Page Source'. Copy and paste all the content in the new tab into this input box.

Select 'All Meals' to view receipts / labels for all meals. Right-click in popup and select 'Save As'. Save the page to your device and use the file upload button below.
